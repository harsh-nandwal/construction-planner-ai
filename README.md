# 🏗️ Construction Planner AI

AI-powered construction planning agent using LangChain + Groq + Streamlit.

## 🚀 Features
- Goal-based planning
- Multi-step execution plan
- Resource validation (budget, engineers, materials, permits)
- Fast LLM using Groq
- Streamlit UI

## 🛠️ Tech Stack
- Python
- LangChain
- Groq API
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
